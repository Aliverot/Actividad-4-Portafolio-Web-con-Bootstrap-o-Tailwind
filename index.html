Sobre Mí & Habilidades

Soy estudiante de Ingeniería en Sistemas Computacionales enfocado en el desarrollo de soluciones integrales. Tengo experiencia en el ciclo de vida del software, desde la planeación y modelado de bases de datos hasta la implementación de interfaces de usuario.

Mis habilidades técnicas incluyen desarrollo web Full-stack (HTML, CSS, JS, PHP, Laravel), programación en C++ (OpenGL, SFML), Python para análisis y simulación, y administración de redes y servidores (Ubuntu Server, pfSense).

Casa de la Cultura

Desarrollo de un sistema de gestión web Full-stack utilizando el framework Laravel (PHP). Este proyecto fue diseñado para administrar el inventario y mantener la trazabilidad de artículos y documentos para la Casa de la Cultura de Cuilápam de Guerrero.

MotoFix

Software de administración desarrollado para un taller de motocicletas. El sistema fue programado utilizando Python e integra una base de datos relacional construida en PostgreSQL para el manejo eficiente de clientes, vehículos y servicios.

Simulación SEIR

Proyecto de análisis matemático y programación en Python. Se implementó el modelo epidemiológico SEIR utilizando el método numérico de Euler para simular y analizar un brote de dengue aplicado a la demografía de Oaxaca de Juárez.

utileria.js

Librería de funciones de utilidad construida puramente con JavaScript (Vanilla JS). Fue diseñada para facilitar procesos repetitivos en el frontend, como la validación estructurada de formularios y el formato de datos sin depender de frameworks externos pesados.
